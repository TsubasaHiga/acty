# バージョニングについて

[セマンティック バージョニング](https://semver.org/lang/ja/)を基本として、以下のルールを適用します。
タグ付けする際も、このルールに従ってタグを付けてください。

## メジャーバージョンアップ

APIの変更に互換性のない場合。

```txt
Ver1.x.x
```

## マイナーバージョンアップ

後方互換性があり機能性を追加した場合。

```txt
Ver1.x.0
```

## パッチバージョンアップ

後方互換性を伴うバグ修正をした場合。

```txt
Ver1.0.x
```

## beta 版

リリース版まではいかない場合。

※ xは任意の連番が入ります。

```txt
Ver1.0.0-beta.x
```
